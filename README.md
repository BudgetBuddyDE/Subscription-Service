# Subscription-Service

## Installation

1. Create an `.env`-file which stored our Supabase credentials like this

    ```
    SUPABASE_URL=
    SUPABASE_ANON_KEY=

    SERVICE_EMAIL=
    SERVICE_PASSWORD=
    ```

2. Install dependencies

    ```shell
    npm i
    ```

3. Start the app

    ```shell
    npm start
    ```

4. Create an build

    ```shell
    npm run build
    ```
